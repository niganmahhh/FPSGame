# FPSGame

Web based 第三人称战术射击新手训练场，使用 Vite + Three.js 构建。

## 运行

```bash
npm install
npm run dev
```

进入页面后点击“进入训练场”，使用鼠标锁定视角。

## 操作

- `WASD` 移动
- `Shift` 冲刺
- `Space` 跳跃
- `Ctrl` / `C` 蹲伏
- 鼠标移动调整视角
- 鼠标左键射击
- `R` 换弹
- `V` 切换第一/第三人称
- `1` / `2` / `4` / `5` 切换步枪
- `3` / `Q` 快速切刀
- `6` 切换第二把刀

## 资产说明

武器和刀具均为 Three.js 程序化原创模型，用于公开发布；未包含《穿越火线》或其他商业 FPS 的专有模型文件。
